# Nonebot_Plugin_QuestionMark
  
_✨ 基于OneBot适配器的[NoneBot2](https://v2.nonebot.dev/)行为艺术插件 ✨_
  
</div>

## 功能

有群友或私聊发送单个问号 `?` 或 `？`, bot会自动回复颠倒的问号 `¿`

## 安装

- 使用 nb-cli

```
nb plugin install nonebot_plugin_questionmark
```

- 使用 pip

```
pip install -U nonebot_plugin_questionmark
```
